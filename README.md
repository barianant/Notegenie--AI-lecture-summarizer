# 📘 NoteGenie – AI Lecture Summarizer

NoteGenie is an AI-powered web application that helps students quickly understand long lectures by generating concise summaries and visual mind maps

### 🚀 Features

-  AI-powered lecture summarization  
-  Converts long lecture content into short notes  
-  Generates visual mind maps for better understanding  
-  Fast and simple interface for students  
-  adjustable summary length
  
## Run Locally
1. Clone the repository:
   `git clone https://github.com/barianant/Notegenie--AI-lecture-summarizer.git`
2. Change Directory:
   ` cd Notegenie--AI-lecture-summarizer`
3. Install dependencies:
   `npm install`
4. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
5. Run the app:
   `npm run dev`
